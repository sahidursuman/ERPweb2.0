报账审核--计算公式

所有模块

删除：已对账、坏账以及产生了收支明细的数据不能删除

坏账：发团安排中删除了安排，报账审核中展示安排数据，并且不能更改可输入框的数据

单价/底价：团的审核状态不为待报账状态，可以更改单价、底价

团款

项目 备注
团款现收 多导：选择导游，默认团款现收状态为已收到
费用调整 调整该组游客的费用价格

购物

项目 公式 备注
导佣 导佣 = 打单金额 x 导佣比例 -
社佣 社佣 = 打单金额 x 社佣比例 多导：社佣 = 金额小计 x 社佣比例
人数返佣 人数返佣 = 金额小计 x 团队人数(大人) -
停车返佣 停车返佣 = 金额小计 x 车辆数 当发团时没有安排车辆，停车返佣为0，当安排了多辆车，按一辆车计算停车返佣
购物小计 购物小计 = 该购物店所有商品打单金额之和 该购物店的人数返佣、停车返佣不计算在内
导佣小计 导佣小计 = 该购物店所有导游返佣金额之和 该购物店的人数返佣、停车返佣不计算在内
社佣小计 社佣小计 = 该购物店所有社佣金额之和 该购物店的人数返佣、停车返佣不计算在内
购物收入 购物收入 = 导游返佣 + 社佣 该购物店的人数返佣、停车返佣参与计算
导游购物返佣 导游购物返佣 = 导游返佣之和 该购物店的人数返佣、停车返佣不计算在内

自费

项目 公式 备注
应收 应收 = 现收 -
应收优惠 应收优惠 = (单价 x 应收数量) - 现收 -
应付 应付 = (底价 x 应付数量) - 应付优惠 -
导佣 导佣 = (现收 - (底价 x 应收数量)) x 导佣比例 -
社佣 社佣 = (现收 - (底价 x 应收数量)) x 社佣比例 -
自费收入 自费收入 = 自费项目的应收之和 -
自费费用 自费费用 = 自费项目的应付之和 -
导佣自费返佣 导佣自费返佣 = 自费项目的导游佣金之和 -

其他收入

项目 公式 备注
金额 金额 = 单价 x 数量 -
金额 金额 = 单价 x 数量 -

车费

项目 公式 备注
车费应付 车费应付 = 车费 - 优惠 -
车费费用 车费费用 = 车费应付列之和 -

餐费

项目 公式 备注
应付 应付 = 金额 x 人数 - 优惠 -
人数合计 人数合计 = 导游实销中人数之和 -
房费费用 房费费用 = 房费应付列之和 -

景区

项目 公式 备注
应付 应付 = 金额 x 数量 - 优惠 -
数量合计 数量合计 = 导游实销中数量之和 -
景区费用 景区费用 = 景区应付列之和 -

票务

项目 公式 备注
应付 应付 = 金额 x 数量 - 优惠 -
数量合计 数量合计 = 导游实销中数量之和 -
票务费用 票务费用 = 票务应付列之和 -

其他支出

项目 公式 备注
应付 应付 = 金额 x 数量 - 优惠 -
数量合计 数量合计 = 导游实销中数量之和 -
其他费用 其他费用 = 其他支出应付列之和 -

保险

项目 公式 备注
应付 应付 = 金额 x 数量 - 优惠 -
保险费用 保险费用 = 保险费用应付列之和 -

导游

项目 公式 备注
导服费 - 导服费 = 导服费列之和
导服管理费 - 导服管理费 = 管理费列之和
